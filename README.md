## Deskripsi
"Bisikan Sufi" adalah portal spiritual yang menuntun pengunjung untuk mendengarkan suara hati, memahami pesan batin, dan menemukan ketenangan batin. Website ini menyajikan:

- Quote harian acak
- Video inspiratif dalam playlist
- Audio meditasi background
- Dark/Light mode
- Tombol share ke sosial media

## Cara Deploy
1. Upload semua file ke repository GitHub (root branch)  
2. Aktifkan GitHub Pages → Branch: main / Folder: root (/)  
3. Tunggu beberapa menit → Website live  

## Roadmap Pengembangan
1. **Stabil & Profesional**: live website, responsive, backup, tampilan rapi  
2. **Interaktif & Engagement**: quote favorit, share, playlist video, audio, animasi  
3. **SEO & Growth**: meta tags, sitemap, newsletter, multi-platform share  
4. **Eksperimen & Fitur Lanjutan**: PWA, komentar pengunjung, dark/light mode otomatis, analytics

## Tools & Library
- CSS & Animasi: TailwindCSS, AOS.js, GSAP  
- Frontend: Vanilla JS / Alpine.js  
- Backend ringan (opsional): Firebase / Supabase  
- SEO: meta tags, Open Graph, sitemap
